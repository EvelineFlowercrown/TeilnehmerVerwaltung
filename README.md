# Teilnehmendenverwaltung
*by [BinaNoir](https://github.com/BinaNoir), [EvelineFlowercrown](https://github.com/EvelineFlowercrown) and [grimmchristoph84](https://github.com/grimmchristoph84-commits)*

## Beschreibung
Diese Datenbank dient zur Verwaltung von Teilnehmenden einer Reha-Maßnahme. 

## Installation
1. Lade Python 3.3.17 (oder neuer) von https://www.python.org/ und installiere es.
2. Öffne deinen Terminal: 
Linux: `[strg] + [alt] + [t]`<br>
Windows: `[win] + [X]` → `wt` → `[Enter]`
3. Klone dieses Repository:
```bash
git clone https://github.com/EvelineFlowercrown/TeilnehmerVerwaltung.git
```
4. Installiere die benötigten Pakete aus requirements.txt :
```bash
pip freeze > requirements.txt
```
